# Crawler - 菜单数据采集工具

本项目是一个专门用于从菜单截图或相关图片中提取数据的采集工具。通过 OCR 技术，将非结构化的图片菜单转换为结构化的 JSON 数据，为后续的数据清洗与分析提供支持。

## 主要功能
- **OCR 识别**：基于 PaddleOCR 实现高精度的文字识别。
- **移动端自动化采集**：支持通过 ADB 自动控制 Android 手机（如 Redmi K70）在美团 App 中进行店家浏览与菜单截图。
- **图像识别辅助**：引入 OpenCV 进行界面元素识别，实现自动化流程控制。
- **结构化输出**：识别结果自动保存为 JSON 格式，包含文本内容及坐标信息，便于后续进行数据清洗与结构化处理。

## 快速开始
1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
2. **运行工具**：
   ```bash
   python test_ocr.py
   ```
3. **使用说明**：
   - 程序启动后，根据提示输入图片路径（例如：`img/menu.png`）。
   - 识别结果将自动保存至 `raw_data/` 目录下。
   - 输入 `exit` 即可退出程序。

## 数据流向
本工具采集的原始数据（JSON）将经过清洗与结构化处理后，最终导入到主项目 `Real_Howmuch` 的数据目录中。
