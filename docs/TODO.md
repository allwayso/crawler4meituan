# 菜品数据采集与清洗方案

## 1. 核心流程
1. **AI 数据识别与清洗**：将 OCR 原始数据直接输入 LLM，利用其语义理解能力，直接提取菜品名、描述和价格，实现结构化清洗。
2. **移动端自动化采集**：使用 ADB 脚本结合 OpenCV 图像识别，实现美团 App 自动进入店家、自动截图菜单页面的流程。
