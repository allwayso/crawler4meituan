# 自动化采集流程文档

## 目标
遍历餐厅列表，对每个餐厅进行深度信息采集。

## 流程步骤
1. **初始化：** 启动应用，进入餐厅列表页。
2. **列表识别：** 截图并识别当前屏幕上的餐厅列表。
3. **遍历循环：**
    *   **选择餐厅：** 选取列表中第一个未访问的餐厅。
    *   **进入详情：** 点击该餐厅进入详情页。
    *   **采集数据：**
        *   点击“菜品”标签。
        *   点击固定位置的“查看全部”。
        *   点击固定位置的“网友推荐”。
        *   截图保存。
        *   滑动屏幕。
        *   截图保存。
    *   **返回列表：** 返回左上角（两次返回操作）。
    *   **标记状态：** 将该餐厅标记为已访问。
4. **列表更新：** 若列表已遍历完，执行滑动操作以加载更多餐厅，重新识别列表，重复步骤 3。
